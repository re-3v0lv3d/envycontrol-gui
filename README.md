# EnvyControl GUI

Una interfaz gráfica moderna y sencilla para **EnvyControl** en Linux.  
Cambia fácilmente entre modos **Integrated**, **Hybrid** y **NVIDIA** con un clic.

<img width="691" height="665" alt="image" src="https://github.com/user-attachments/assets/5425dbcc-004d-4034-b71b-7fe501edfdbe" />


## Características

- Modo claro y oscuro automático
- Soporte para opciones avanzadas (`--rtd3`, `--force-comp`, `--coolbits`)
- Icono en la bandeja del sistema (System Tray)
- Uso de `pkexec` (no necesitas abrir terminal)
- Confirmación antes de reiniciar
- Detecta automáticamente el modo actual

## Requisitos

- Python 3.9 o superior
- [EnvyControl](https://github.com/bayasdev/envycontrol) instalado
- Drivers NVIDIA propietarios
- `pkexec` / polkit configurado
